# **App Name**: NEU MOA Tracker

## Core Features:

- User Authentication: Secure user login using institutional Google-based email accounts.
- Role-Based Dashboards: Custom dashboards for Admin, Faculty, and Student roles with appropriate access levels and data visibility.
- MOA Management: Functionality to add, edit, and soft delete MOA records, with recovery option for Admins.
- MOA Status Tracking: Track MOAs through various statuses: APPROVED, PROCESSING, EXPIRED, and EXPIRING.
- Audit Trail: Record user actions (insert, edit, delete) with timestamps, visible to Admins for compliance and accountability.
- Statistics Dashboard: Display key statistics (active, processing, expired MOAs) via dashboard cards, filterable by college and date range.
- Intelligent Search: Enable a search tool to find MOAs based on various criteria, enhanced with AI to suggest search terms and relevant MOAs based on partial input, boosting findability and ease of use.

## Style Guidelines:

- Primary color: Deep teal (#008080) for a professional and trustworthy feel.
- Background color: Light teal (#E0F8F7), creating a soft and clean backdrop.
- Accent color: Yellow-green (#B0E293) to highlight key actions and information, ensuring critical elements stand out.
- Body and headline font: 'Inter' sans-serif for a modern, neutral and objective look.
- Consistent use of minimalistic icons to represent MOA statuses and actions, providing clear visual cues.
- Clean, intuitive layout with clear visual hierarchy to facilitate easy navigation and information processing for all user roles.
- Subtle transitions and loading animations to enhance user experience without being distracting.
